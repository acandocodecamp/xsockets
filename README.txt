************************************************
* Vad är XSockets
************************************************

- XSockets är ett realtidsramverk som möjliggör meddelandebaserad kommunikation mellan alla enheter som har TCP/IP.
- Full-duplex kommunikation -> websockets
- Halv-duplex -> ajax

************************************************
* Jämförerlse med SignalR
************************************************

Websockets överallt, inte bara .NET 4.5 på Windows Server 2012+ or Windows 8+
Pub/Sub (går att bygga ovanpå SignalR)
Offlinemeddelanden
State på servern (SignalR: Persistent connection, groups)

http://xsockets.net/xsockets-vs-signalr
